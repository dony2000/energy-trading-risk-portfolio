#### \# Energy Trading, Risk \& Optimisation Portfolio

#### 

#### This repository contains practical projects and short market notes to build job-ready skills for energy trading and risk management.

#### 

#### \## Structure

#### \- `notes/` — weekly 1-page market notes (PDF/Markdown)

#### \- `notebooks/` — Python analysis notebooks (pandas, charts, risk metrics)

#### \- `data/` — small datasets used in the notebooks (cleaned)

#### \- `outputs/` — charts and exported tables for notes/reports

#### 

#### \## How to run

#### Open the notebooks in Google Colab or Jupyter Notebook and run cells top-to-bottom.
